# Angular Workshop

### Run

```
npm install
```

```
bower install
```

### Build

```
gulp
```

### Release

```
gulp --release
```

### Backend (Rails)

Create a new app

```
rails new app -T -j postgres

// Scaffold a new model
rails g scaffold post title body:text

// Run db migrate
rake db:migrate
```
